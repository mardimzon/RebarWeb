Connection Troubleshooting

If you're having trouble connecting to your Raspberry Pi:

  • Make sure your Raspberry Pi is powered on and the RebarVista application is running
  • Verify that you've connected to the "RebarVista" WiFi network on your device
  • The default WiFi password is "rebarconnect"
  • If you still can't connect, try restarting your Raspberry Pi
  • For external camera setup, make sure the camera is properly connected before starting the application

External Camera Information

This application has been updated to work with external cameras:

  • Connect your USB camera to the Raspberry Pi before starting
  • The camera index can be configured in the settings page
  • Default index is 0, which is typically the first connected camera
  • If you have multiple cameras, you may need to try different indices (0, 1, 2, etc.)
